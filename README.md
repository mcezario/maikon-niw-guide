# EB2 NIW Educative Website

A static site that educates recommenders about Maikon's EB2 NIW process.

**Stack:** Vite + React + TypeScript + Tailwind CSS

---

## Project Structure

```
eb2-educative/
└── client/   # React frontend (Vite)
```

---

## Prerequisites

- Node.js 18+
- npm 9+

---

## Setup

### 1. Install dependencies

```bash
cd client && npm install
```

---

## Running the Project

### Development

```bash
cd client && npm run dev
```

- Client: http://localhost:5173

### Production build

```bash
cd client && npm run build
```

The output is in `client/dist/` — deploy it to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.).
